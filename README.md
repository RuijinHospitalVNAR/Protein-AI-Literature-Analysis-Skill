# Protein AI Literature Analysis Skill

## Overview

Protein AI Literature Analysis Skill is a comprehensive tool for retrieving and analyzing the latest scientific literature on AI-driven protein design. It automates the process of searching reputable sources, classifying research areas, evaluating significance, and generating structured reports with direct links to original papers.

## Features

### 1. Literature Retrieval
- **Multi-source search**: PubMed, bioRxiv, arXiv, Google Scholar, and X (Twitter)
- **Comprehensive coverage**: Captures both peer-reviewed papers and preprints
- **Advanced filtering**: Focused on AI & protein design research

### 2. Research Classification
- **Detailed categories**: Deep Learning, Generative Models, Reinforcement Learning, etc.
- **Automated classification**: Based on keyword extraction and topic modeling
- **Hierarchical organization**: Structured research areas for better navigation

### 3. Importance Scoring
- **Multi-dimensional scoring**: Novelty (30%), Impact (25%), Technical Innovation (25%), Applicability (20%)
- **Automated evaluation**: Contextual scoring based on content analysis
- **Ranking system**: Papers sorted by importance score

### 4. Interactive Dashboard
- **Real-time visualization**: Category distribution and publication timeline
- **Advanced filtering**: By category, time period, and source
- **Search functionality**: Full-text search across all papers
- **Responsive design**: Works on desktop and mobile devices

## Getting Started

### Option 1: Local Development (with Backend)

1. **Clone the repository**
   ```bash
   git clone https://github.com/RuijinHospitalVNAR/Protein-AI-Literature-Analysis-Skill.git
   cd Protein-AI-Literature-Analysis-Skill
   ```

2. **Install dependencies**
   ```bash
   pip install flask
   ```

3. **Run the backend server**
   ```bash
   python web/backend/app.py
   ```

4. **Access the dashboard**
   Open your browser and go to `http://127.0.0.1:5000`

### Option 2: Static Frontend (GitHub Pages)

1. **Visit the GitHub Pages site**
   Go to `https://ruijinhospitalvnar.github.io/Protein-AI-Literature-Analysis-Skill`

2. **Automatic updates**
   The site is automatically updated weekly with the latest literature

## Usage

### Updating Literature Data

#### Local Update
1. Run the update script
   ```bash
   python web/backend/update_script.py
   ```

#### Automated Update (GitHub Actions)
The repository is configured with GitHub Actions to automatically update the literature data weekly. No manual intervention is required.

### Customizing the Skill

1. **Modify search parameters**
   Edit `scripts/literature_retriever.py` to adjust search terms, timeframes, and sources

2. **Update classification schema**
   Modify the classification categories in `scripts/paper_analyzer.py`

3. **Adjust scoring criteria**
   Update the scoring weights in `scripts/paper_analyzer.py`

## API Endpoints

When running the backend server, the following API endpoints are available:

- `GET /api/papers` - Returns all retrieved papers
- `GET /api/analyzed` - Returns analyzed papers with classifications and scores
